This folder contains implementatons for TTK4900 Master's Thesis


The files run<controller>.m runs the simulations.
The files sim<controller>.xls contains the Simulink models.
The files plots<controller>.m generates plots.
 
The controllers are abbreviated:
AIDC - Adaptive Inverse Dynamics Control
STA - Super-Twisting Algorithm with adaptive gains
NRAC - Non-Regressor based adaptive control
IC - Impedance Control
FC - PI Force Control with impedance control

The folder also contains the script provided by Henrik Schmidt-Didlaukies:
